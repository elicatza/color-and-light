# Hexhsl
A program that converts RGB colors in hexadecimal to Hue Saturation Luminance (HSL).

<br>

## Dependecies
- gcc
- glibc

<br>

## Install
```sh
git clone "https://github.com/elicatza/hexhsl"
cd hexhsl
make
sudo make install
```

<br>

## Uninstall
```sh
git clone "https://github.com/elicatza/hexhsl"
cd hexhsl
sudo make uninstall
```
