# Color and Light
A program that converts color formats to others.

## Dependecies
- gcc
- glibc

## Install
```sh
git clone "https://github.com/elicatza/hexhsl"
cd hexhsl
make
sudo make install
```

## Uninstall
```sh
git clone "https://github.com/elicatza/hexhsl"
cd hexhsl
sudo make uninstall
```
