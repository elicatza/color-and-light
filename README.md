# Color and Light
A program that converts HEX colors to HSL and HSV.

## Dependecies
- gcc
- glibc

## Install
```sh
git clone "https://github.com/elicatza/color-and-light"
cd color-and-light
make
sudo make install doc
```

## Uninstall
```sh
git clone "https://github.com/elicatza/color-and-light"
cd color-and-light
sudo make uninstall
```
